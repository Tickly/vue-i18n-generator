# vue-i18n-generator

如果你也这样
- 每次`新增`/`修改`一个Key，都要到不同的`语言`下面，小心翼翼的确定`层级`是否正确，拼写是否有误。
- 有时候想看某个key的所有语言内容做个对比。

那么可以试试这个工具。

支持导入，导出。
